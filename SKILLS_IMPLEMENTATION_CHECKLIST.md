## 🔄 Maintenance Guide

### Adding New Skills
1. Edit `/src/data/skills.js`
2. Add skill object to category
3. Add icon to `iconMap` if needed
4. Rebuild and deploy

### Customizing Colors
1. Edit category color in skills.js
2. Update icon colors if needed
3. Rebuild and deploy

### Adjusting Animations
1. Edit transition values in components
2. Test smoothness
3. Rebuild and deploy

### Fixing Issues
1. Check browser console for errors
2. Verify skill data structure
3. Check icon names
4. Verify color format
5. Test on mobile device
