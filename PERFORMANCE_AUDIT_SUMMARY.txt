================================================================================
CRITICAL ISSUES (FIX FIRST)
================================================================================

1. OceanBackground: Expensive Background Animations
   - 30 particles with infinite animations
   - 5+ layers with scroll-based updates (no throttling)
   - Multiple mix-blend-modes = expensive compositing
   - FIX: Reduce particles to 8-20, add RAF throttling, reduce blend-modes
   - IMPACT: 25-35% FPS improvement
   - TIME: 2-3 hours

2. ContactCard: Mousemove Causes Re-render Cascade
   - 200+ state updates per second on hover
   - No debouncing/throttling
   - Perspective transforms recalculated each update
   - FIX: Use CSS variables instead of React state
   - IMPACT: 50-60% FPS improvement (most critical)
   - TIME: 1-2 hours

3. useMousePosition & SpotlightBackground: Unthrottled Global Listeners
   - EVERY mousemove fires state update
   - Affects entire page re-renders
   - DOM style manipulation 200+ times/sec
   - FIX: Add 16ms throttling to mousemove handlers
   - IMPACT: 40-50% FPS improvement
   - TIME: 1-2 hours

================================================================================
HIGH PRIORITY ISSUES
================================================================================

4. Missing React.memo on List Components
   - ContactCard not memoized (re-renders with parent)
   - InteractiveSkillCard not memoized
   - ProjectCard3D not memoized
   - FIX: Wrap with React.memo()
   - IMPACT: 15-25% fewer renders on page transition
   - TIME: 1 hour

5. Multiple setState Calls in Navbar Scroll Handler
   - Each setState triggers separate render pass
   - FIX: Batch into single state object update
   - IMPACT: 10-15% fewer renders on scroll
   - TIME: 1-2 hours

6. Contact Page Particle Animations
   - 15 simultaneous infinite animations
   - Different durations = 45+ frame updates/sec combined
   - FIX: Reduce to 5-8 particles, use CSS animations
   - IMPACT: 20-30% FPS improvement on that page
   - TIME: 1 hour

7. CSS Performance Issues
   - 66 kB CSS file (1967 lines)
   - Multiple expensive box-shadows (3 shadows per element)
   - Heavy blur filters (blur: 60px, blur: 20px on many elements)
   - Unused animations and styles
   - FIX: Consolidate shadows, reduce blur on mobile, remove unused CSS
   - IMPACT: 15-20% paint time reduction
   - TIME: 3-4 hours

================================================================================
MEDIUM PRIORITY ISSUES
================================================================================

8. Missing useCallback on Event Handlers
   - InteractiveContactForm handlers recreated on every render
   - ProjectCard3D handlers missing useCallback
   - FIX: Add useCallback() wrapper
   - IMPACT: 5-10% fewer re-renders
   - TIME: 1 hour

9. Prop Drilling & Unused Props
   - ContactCard receives unused "showAvailability" prop
   - Long prop chains create deep re-render trees
   - FIX: Remove unused props, use composition
   - IMPACT: 5-8% render time improvement
   - TIME: 30 minutes

10. Missing Image Optimization
    - Hero image not optimized
    - No lazy loading attributes
    - No responsive image srcsets
    - FIX: Optimize, add loading="lazy", create srcsets
    - IMPACT: 10-20% LCP improvement
    - TIME: 1-2 hours

================================================================================
LOW PRIORITY ISSUES
================================================================================

11. Bundle Size Optimization
    - Lucide icons could be tree-shaken better
    - Three.js unused (if not used)
    - FIX: Tree-shake imports, lazy-load animations CSS
    - IMPACT: 5-10% bundle reduction
    - TIME: 1 hour

12. Accessibility: Reduced Motion Support
    - Already has prefers-reduced-motion in CSS
    - Should disable particle animations for this group
    - FIX: Add JavaScript checks for reduced motion
    - IMPACT: Inclusive experience for 15% of users
    - TIME: 30 minutes
