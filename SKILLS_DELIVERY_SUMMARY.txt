================================================================================
                        QUICK START GUIDE
================================================================================

TO VIEW:
   1. Navigate to /skills in your portfolio
   2. Click category tabs to filter
   3. Click view mode buttons to switch
   4. Hover over cards for interactions

TO ADD A SKILL:
   1. Edit /src/data/skills.js
   2. Add: { name: "Skill", level: 85, icon: "icon-key" }
   3. Run: npm run build
   4. Done!

TO CHANGE COLORS:
   1. Edit /src/data/skills.js
   2. Change: color: "#new-color"
   3. Run: npm run build
   4. Done!

TO ADD NEW ICON:
   1. Edit /src/components/ui/InteractiveSkillCard.jsx
   2. Add to iconMap: icon-key: "emoji"
   3. Use in skills.js
   4. Done!
